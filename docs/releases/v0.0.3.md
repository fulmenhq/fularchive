# refbolt v0.0.3

**Release date**: 2026-04-02

**Windows ARM64, Scoop support, and `make install`.**

## Highlights

- **6-platform builds**: Added `windows/arm64` cross-compiled binary. All platforms built from single `ubuntu-latest` runner with `CGO_ENABLED=0`.
- **`make install`**: Build + copy to `~/.local/bin` (INSTALL_DIR overridable). No more manual copy after `make build`.
- **Scoop (Windows)**: Manifest in `fulmenhq/scoop-bucket` with both x64 and ARM64 support.
- **CI/CD documentation**: Full cross-compilation rationale, 6-platform build table, package distribution channels, and migration notes for future CGO needs.

## Build Targets

| Platform      | GOOS    | GOARCH | Binary Name                 | New |
| ------------- | ------- | ------ | --------------------------- | --- |
| Linux x64     | linux   | amd64  | `refbolt-linux-amd64`       |     |
| Linux ARM64   | linux   | arm64  | `refbolt-linux-arm64`       |     |
| macOS x64     | darwin  | amd64  | `refbolt-darwin-amd64`      |     |
| macOS ARM64   | darwin  | arm64  | `refbolt-darwin-arm64`      |     |
| Windows x64   | windows | amd64  | `refbolt-windows-amd64.exe` |     |
| Windows ARM64 | windows | arm64  | `refbolt-windows-arm64.exe` | ✅  |
